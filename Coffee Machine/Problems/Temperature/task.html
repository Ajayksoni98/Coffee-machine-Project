Temperature Open on JetBrains Academy

Below you can see a draft of a program for a basic temperature converter from Fahrenheit to Celsius and vice versa. You need to finish the program by creating a sort of entry point for the program in the function main.

In this function, you should:

  • process the input. The input is a string containing a temperature value and a unit of measurement separated by a whitespace character. We already wrote the code that reads the input and creates the needed variables, but you need to convert them to the right types.
  • call the necessary function depending on what was given in the input
  • print the converted temperature with the new unit

Use the examples as a guide.

You do NOT need to call the function main.


Sample Input:
451 F

Sample Output:
232.78 C

Sample Input:
-32 C

Sample Output:
-25.6 F

Sample Input:
-40 F

Sample Output:
-40.0 C

Sample Input:
41 C

Sample Output:
105.8 F
